# The idea behind the project

The idea of this project is to build simple but effective alram system for fire
while keeping in mind about low-power consumation and easy to use.

# Working idea of the project

The working idea of the project is that if the fire is detected by the system it will
then using the buzzer alram the people and using servo motor open the doors for the people to get outside of the building.
This will also be used to sent notification to fire department for any case of fire break out.
The system is configured for detecting two things which is smokes and gases which may lead to fire and secondly
fire itself.

# Parts used

- Arduino-UNO -> for the main controller
- MQ-2, MQ-9, MQ-135 -> for gases detection
- OLED display -> to show wheather its a fire or smoke
- buzzer for alram systems 
- servo motor -> for opening and closing the doors
- 16 channel servo motor driver -> for controlling the servo motor that are used in the project
